## demo_diver

The `DemoDivider` widget provides a simple wave divider that you can customize. Here are some usage examples:

1. Using the default divider:
```dart
   
DemoDivider()
    
```

or a little bit of custom with:
```dart 
    DemoDivider(
      height: 3,
      dividerColor: Colors.amber,
      thickness: 7,
    )
```
